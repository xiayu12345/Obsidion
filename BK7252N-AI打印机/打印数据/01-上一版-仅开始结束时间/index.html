    组一：仅打印开始 / 结束时间（上一版全局参数）¶

    纸上应出现的内容¶

    仅三行测速信息，例如：

    S:12345
E:14123
C:1778ms

    不应出现 Modes: 及六种模式名列表。

    用途¶

    • 与「本版」样张对比时，可排除「模式列表占用少量走纸/文字时间」的差异，主要看同一套图像在旧全局参数下的 C 值。

    上一版（无 Modes 页脚时）speed=5 全局参数纪要¶

    以下为与当前仓库对比的「上一版」快照，对应 thermal_printer_print_bitmap 内仍只有 S/E/C 三行、尚未调用 lv_desktop_format_print_footer 时的参数量级。

    项 数值
    inter_group_us 1
    line_cooling_us 12
    heat_us_at_5c / 25c / 40c 1120 / 980 / 860
    motor_step_us 680
    status_check_period_lines 80
    连续白行批量上限 64
    周期性 rt_thread_mdelay(1) 约每 128 行
    位图结束后 thermal_printer_feed 2
    高密度行额外加热（黑点档） +170 / +115 / +70 / +38 / +14（分档）

    归档建议¶

    • 文件名建议：YYYYMMDD_模式名_仅SEC_备注.jpg（模式名仅作你当时 UI 记录，该版纸上不打印模式）。
