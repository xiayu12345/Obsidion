# 打印数据归档说明

本目录将**样张 / 测速记录**按固件版本分为两组，便于对比分析。

| 子目录 | 含义 | 纸上特征 | 对应固件 |
|--------|------|----------|----------|
| [01-上一版-仅开始结束时间](%E6%89%93%E5%8D%B0%E6%95%B0%E6%8D%AE/01-%E4%B8%8A%E4%B8%80%E7%89%88-%E4%BB%85%E5%BC%80%E5%A7%8B%E7%BB%93%E6%9D%9F%E6%97%B6%E9%97%B4.md) | 仅测速三行 | 只有 `S` / `E` / `C`，**无** `Modes:` 列表 | 上一版编译（全局参数见该目录 README） |
| [02-本版-含模式状态](%E6%89%93%E5%8D%B0%E6%95%B0%E6%8D%AE/02-%E6%9C%AC%E7%89%88-%E5%90%AB%E6%A8%A1%E5%BC%8F%E7%8A%B6%E6%80%81.md) | 测速 + 模式 | `S` / `E` / `C` 下方有 `Modes:` 与六种模式，当前行为 `* ` 前缀 | **当前**仓库编译（全局参数见该目录 README） |

请将实拍照片或扫描件放入对应子目录（或在此 README 同层按日期建子文件夹亦可）。
