本工作区按**项目**组织笔记。每个项目下有开发记录、Bug 追踪、SDK 说明等子域。

**目录约定**：笔记按文件夹分层存放，避免全部堆在 vault 根目录。Dendron 层级仍由**文件名**（点号分隔）决定，wikilink 无需改动。

```
项目/
  项目.md                          ← 本页
  <项目标识>/
    项目.<项目标识>.md             ← 项目首页
    开发记录/
      日志/   指南/
    Bug追踪/  打印数据/  芯片SDK/  总结/
```

## 添加新项目

1. 在 `项目/<项目标识>/` 下新建 `项目.<项目标识>.md` 作为项目首页。
2. 子笔记放入对应子文件夹（如 `开发记录/日志/`），文件名仍用 `项目.<项目标识>.<子域>.<笔记名>.md`。
3. 在本页表格中增加一行入口链接。
4. 在 `root.schema.yml` 中注册新项目。
