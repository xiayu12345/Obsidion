# H133 硬件资料

硬件资料按“主控板”和“外设”分开整理，PDF/图片附件放在 `H133/hardware/` 下，发布到 MkDocs 后可直接点击查看。

## 分类入口

- [主控板](主控板/index.md)
- [外设](外设/index.md)

## 附件目录

| 目录 | 用途 |
|---|---|
| `hardware/datasheets/` | 芯片、屏幕、触摸、存储、多媒体等数据手册 |
| `hardware/schematics/` | 主控板原理图 |
| `hardware/pcb/` | PCB 顶层、底层、丝印、3D 图 |
| `hardware/photos/` | 实物照片、接线照片、修改记录照片 |

## 当前已集成附件

| 分类 | 数量 | 说明 |
|---|---:|---|
| 主控板 | 2 | H133 Datasheet、拓步 10.1 寸 H133 主板原理图 |
| 外设 | 7 | 触摸、屏幕、SPI NOR、视频解码库资料 |

## 维护约定

- 主控板相关：放入 [主控板](主控板/index.md)
- 屏幕、触摸、存储、音视频等外设：放入 [外设](外设/index.md)
- 新增 PDF/图片后，先放到 `hardware/` 对应目录，再在分类页增加链接
