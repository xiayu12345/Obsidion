## 布局查阅建议

排查硬件问题时对照：

1. SPI NOR 走线（PC2–PC7）
2. SD 卡 TF 座（PF0–PF6）
3. MIPI FPC 座
4. HDMI 座
5. 触摸 FPC（I2C）
6. 功放 / 电源 12V / USB 供电切换

LCD 背光供电分析见 Ubuntu 文档 `H133-LCD背光亮度-硬件供电分析.md`。
