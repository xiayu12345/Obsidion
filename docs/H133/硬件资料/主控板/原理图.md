## 与软件配置对照

| 硬件模块 | 软件配置 |
|---|---|
| SPI NOR | `sys_config.fex` 中 `[spinor_para]` |
| SD 卡 | `sys_config.fex` 中 `[card0_boot_para]` + DTS mmc0 |
| MIPI LCD | `board.dts` panel 节点 |
| HDMI | `board.dts` hdmi 节点 |
| UART0 调试 | `sys_config.fex` 中 `[uart_para]` |

引脚速查见 [引脚定义速查](../../引脚定义速查/index.md)。
