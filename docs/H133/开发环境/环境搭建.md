# H133 开发环境搭建

## 1. 开发机要求

| 项 | 要求 |
|---|---|
| 系统 | Ubuntu 18.04 x86_64（当前：`h133-ubuntu`） |
| 磁盘 | 建议 ≥ 100G 可用（全量编译约 8G+ out） |
| 内存 | 建议 ≥ 8G |
| 网络 | 首次编译需下载 toolchain / dl 包 |

## 2. 获取工程

工程已在 Ubuntu 开发机：

```bash
# Windows 侧 SSH 进入
ssh h133-ubuntu
cd ~/Workspace/HDMI_new/HDMI
```

## 3. 加载构建环境

```bash
cd ~/Workspace/HDMI_new/HDMI
source build/envsetup.sh
# p1_nor 一般无需额外 lunch
```

成功提示：

```
NOTE: The SDK(...) was successfully loaded
load openwrt... ok
load bsp...ok
```

## 4. 常用编译命令

| 命令 | 作用 |
|---|---|
| `m` | 编译（OpenWrt + 应用） |
| `p` | 打包镜像 |
| `./build.sh kernel bootloader` | 改 DTS/内核/U-Boot 后必须先执行 |
| `./build.sh` | 全量编译 shorthand |

**改 board.dts / uboot-board.dts / 内核后的顺序：**

```bash
./build.sh kernel bootloader
m
p
```

## 5. Windows 侧协作

| 方式 | 用途 |
|---|---|
| `ssh h133-ubuntu` | 远程编译、查日志 |
| SFTP / WinSCP | 传文件（板端 OpenSSH 端口 22，root/tina） |
| `adb push` | 快速刷单个 `lv_projector` 二进制 |
| Obsidian（C:\Career） | 笔记编辑 |

## 6. 板端远程调试

1. 板子联网，PC `ssh root@<板子IP>`（密码 `tina`）
2. 或用 SFTP 传 `/usr/bin/lv_projector`
3. 快速验证：

```bash
chmod 777 /usr/bin/lv_projector
/usr/bin/lv_projector
```

- `H133开发文档-wjz/基础资料/lv_projector_烧录流程_123版.md`

## 7. 依赖与常见问题

- **必须在工程顶层**执行 `m` / `p`，不要 `cd openwrt/openwrt && make`（`LICHEE_*` 变量为空会导致 G2D ABI 错乱）
- 首次编译耗时长（1h+），后续增量较快
- toolchain 位于 `out/toolchain/`，缺失时重新 `source build/envsetup.sh && m`
