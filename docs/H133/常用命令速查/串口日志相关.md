# H133 串口日志相关

## 调试串口 UART0

| 项 | 值 |
|---|---|
| 控制器 | uart0 |
| TX | PB08 |
| RX | PB09 |
| 配置来源 | `sys_config.fex` → `[uart_para]` |
| 设备树 | `board.dts` → `&uart0` |

## PC 连接

1. USB 转 TTL 接 PB8(TX) / PB9(RX) / GND
2. 波特率通常 **115200**（以 U-Boot 打印为准）
3. Windows：PuTTY / MobaXterm；Linux：`minicom -D /dev/ttyUSB0`

## 日志层级

| 阶段 | 内容 |
|---|---|
| U-Boot | 启动、DRAM、加载 kernel |
| Kernel | 驱动 probe、mmc、display、G2D |
| Userspace | init → sshd → lv_projector printf |

## 应用日志

`lv_projector` 使用 `printf` 输出到 console，可在串口终端直接查看。

过滤示例：

```bash
# 若已 SSH 进板子，也可看 dmesg
dmesg -w
```

## 其他 UART

| UART | 用途 | 引脚（p1_nor DTS） |
|---|---|---|
| uart0 | 主调试 | PB8/PB9 |
| uart1 | 通用 / DSP debug | 见 board.dts |
| uart2 | 扩展 | 见 board.dts |
| uart3 | P15 排针 | PG0(TX) / PG1(RX) |

> uart3 与 sdc1 互斥，使用 PG0/PG1 时 sdc1 须 disabled。
