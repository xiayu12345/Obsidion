# H133 烧录相关命令

## 整包烧录（推荐量产/大改）

产物：

```
out/h133_linux_p1_nor_uart0_nor.img
```

使用全志 PhoenixSuit / LiveSuit 或 `tools/` 下打包工具，选择 SPI NOR + p1_nor 分区表烧录。

## 单应用快速刷机（开发调试）

### 方式 A：adb push

1. 板子 USB 连接 PC，确认 adb 设备
2. 编译产物推到板端：

```bash
# Windows CMD 示例
adb push out/h133/p1_nor/.../lv_projector /usr/bin/
adb shell chmod 777 /usr/bin/lv_projector
adb shell /usr/bin/lv_projector
```

### 方式 B：SFTP / scp

```bash
# PC → 板子（板子 IP 示例 192.168.1.x）
scp lv_projector root@192.168.1.x:/usr/bin/
ssh root@192.168.1.x "chmod 777 /usr/bin/lv_projector && killall lv_projector; /usr/bin/lv_projector &"
```

板端 SSH：用户 `root`，密码 `tina`，端口 22。

### 方式 C：SD 卡

将二进制拷到 SD 卡，板端 mount 后 cp 到 `/usr/bin/`。

## 验证 USB 调试口

板端：

```bash
cat /sys/devices/platform/soc@3000000/soc@3000000:usbc0@0/usb_device
```

## 注意

- SPI NOR 整包烧录会覆盖 boot，操作前确认镜像与板型匹配（p1_nor）
- 仅改应用时优先 adb/scp，避免全量烧录耗时
