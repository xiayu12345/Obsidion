# H133 编译相关命令

> 均在 Ubuntu 工程根目录执行：`~/Workspace/HDMI_new/HDMI`

## 环境加载

```bash
cd ~/Workspace/HDMI_new/HDMI
source build/envsetup.sh
```

## 日常编译

```bash
m          # 编译（OpenWrt + lv_projector 等）
p          # 打包镜像
```

## 改内核 / U-Boot / DTS 后

```bash
./build.sh kernel bootloader
m
p
```

## 仅编应用

```bash
# 仍在顶层
m package/lv_projector/compile V=s
# 或
make -C openwrt/openwrt package/lv_projector/compile V=s  # 需已 source envsetup
```

## 清理

```bash
# 清理 lv_projector build_dir（G2D 头文件缓存问题时）
rm -rf out/h133/p1_nor/openwrt/build_dir/target/lv_projector/
m
```

## 产物路径

| 产物 | 路径 |
|---|---|
| lv_projector 二进制 | `out/h133/p1_nor/openwrt/build_dir/target/lv_projector/lv_projector` |
| 整包烧录镜像 | `out/h133_linux_p1_nor_uart0_nor.img` |
| rootfs | `out/h133/p1_nor/openwrt/` 下 staging |

## 铁律

1. **必须在工程顶层** `make` / `m`，不要单独进 `openwrt/openwrt/`
2. 改 DTS → 先 `./build.sh kernel bootloader`
3. G2D 异常 → 检查 build_dir 是否旧缓存

## 并行编译

```bash
m -j$(nproc)
```
