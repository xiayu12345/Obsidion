# H133 硬件接线

> 基于 p1_nor 板型与拓步 10.1 寸方案整理，详细以原理图为准。

## 调试接口

| 接口 | 连接 | 说明 |
|---|---|---|
| UART0 | USB-TTL ? PB8(TX) PB9(RX) GND | 115200，U-Boot + kernel + app 日志 |
| USB OTG | Micro/USB-C ? PC | adb 刷应用、供电 |

## 存储

| 接口 | 引脚 | 说明 |
|---|---|---|
| SPI NOR | PC2–PC7 | 板载启动 Flash，无需外接 |
| SD 卡槽 | PF0–PF5 | 四线 SD，挂载 `/mnt/SDCARD` |
| SD CD | PF6 | 卡检测（DTS cd-gpios） |

## 显示输出

| 接口 | 说明 |
|---|---|
| MIPI DSI | 10.1 寸 LCD 面板（拓步） |
| HDMI | 外接显示器/投影 |

背光供电与 USB/12V 关系见 Ubuntu 文档 `H133-LCD背光亮度-硬件供电分析.md`。

## 触摸

| 接口 | 说明 |
|---|---|
| GT911 | I2C（TWI0），中断/复位 GPIO 见 board.dts |
| USB 鼠标 | 可与触摸并存，evdev 双输入 |

## 音频

| 接口 | 说明 |
|---|---|
| SPDIF / I2S | 至功放板 |
| 功放 UART | 独立串口控制音量/静音（见功放调试文档） |

## 网络

| 接口 | 说明 |
|---|---|
| WiFi 模组 | USB 或 SDIO（依硬件版本） |
| 以太网 | 若有 RJ45，见 DTS |

## 扩展串口

| 接口 | 引脚 | 说明 |
|---|---|---|
| UART3 排针 | PG0(TX) PG1(RX) | 与 sdc1 互斥 |
