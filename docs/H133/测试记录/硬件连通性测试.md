## 测试范围

| 项目 | 说明 |
|---|---|
| 串口 | UART0 调试口输出、输入、波特率 |
| ADB/USB | 设备识别、shell、文件传输 |
| SD 卡 | 插拔识别、挂载、读写 |
| LCD/背光 | 屏幕点亮、背光控制 |

| HDMI | 插拔、分辨率、显示输出 |
| 音频 | 功放、SPDIF/I2S、播放 |

```bash
adb shell "df -h"
```
