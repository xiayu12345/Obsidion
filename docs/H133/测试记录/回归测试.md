# 回归测试

## 测试范围

回归测试用于记录缺陷修复后的复测，重点关联 [踩坑记录](../踩坑记录/index.md)、开发记录和具体提交/镜像版本。

| 项目 | 说明 |
|---|---|
| 问题复测 | 已修复问题是否复现 |
| 关联功能 | 修复是否影响周边功能 |
| 镜像对比 | 修复前后镜像行为差异 |
| 长稳验证 | 修复后长时间运行稳定性 |
