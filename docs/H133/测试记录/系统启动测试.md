# 系统启动测试

## 测试范围

| 项目 | 说明 |
|---|---|
| Boot0/U-Boot | 启动阶段串口日志、存储识别 |
| Kernel | 内核启动、设备树加载、驱动初始化 |
| rootfs | 分区挂载、文件系统完整性 |
| 服务启动 | OpenWrt init/procd 服务 |
| 版本确认 | 镜像版本、应用版本、配置文件 |
