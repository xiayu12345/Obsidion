## 关联模块

| 模块 | 说明 |
|---|---|
| TPlayer | 视频播放入口 |
| CedarX | 多媒体底层组件 |
| Display/G2D | 视频输出、旋转、合成链路 |
| LVGL UI | 与视频层共存时需要处理层叠关系 |

## 当前关注点

- 视频旋转在 TPlayer、G2D、display layer 中分别由谁负责。
- HDMI 输出和 LCD 输出是否使用同一套旋转策略。
- UI 与视频共存时 layer 顺序、透明度、格式是否一致。
