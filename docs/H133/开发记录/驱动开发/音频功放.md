## 自动识别到的音频接口

| 接口 | 引脚线索 | 备注 |
|---|---|---|
| I2S0 | PB29/PB23-PB28/PB22 | 需结合原理图确认 |
| I2S1 | PB29/PB23-PB28 | 需结合原理图确认 |
| I2S2 | PF6/PF3/PF5/PF1/PF0 | 与 SDC0 部分引脚复用 |
| SPDIF | PG18 | 功放资料中有 SPDIF 线索 |
| DMIC | PB12/PB11/PB10/PE14/PB8 | 与 LCD/TWI0/UART0 等存在复用风险 |
