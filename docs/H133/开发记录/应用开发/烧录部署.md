## 应用级替换验证

```bash
adb push ./lv_projector /usr/bin/lv_projector
adb shell 'chmod 777 /usr/bin/lv_projector && /usr/bin/lv_projector'
```

## 整包构建

```bash
cd /home/xiayu/Workspace/HDMI_new/HDMI
source build/envsetup.sh
m -j$(nproc)
p
```

## 注意

应用级替换只能验证 `/usr/bin/lv_projector` 等用户态程序修改；DTS、U-Boot、kernel、pinmux、显示驱动变更需要重建并重新烧录完整镜像。
