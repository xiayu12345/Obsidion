# H133 硬件平台

> 板型：**p1_nor** · 配置来源：`sys_config.fex` + `board.dts`（Ubuntu 工程实测）

## SoC 概要

| 项 | 规格 |
|---|---|
| 型号 | Allwinner H133 |
| 内核 compatible | `allwinner,h133`, `arm,sun8iw20p1` |
| CPU | ARM Cortex-A7 |
| 显示引擎 | DE（双屏通道：主显 4ch + 副显 2ch） |
| 2D 加速 | G2D（与 sunxifb / lv_projector 强相关） |
| 典型 Flash | BY25FQ256ES 类 **32MB SPI NOR** |

## 存储与分区

### SPI NOR 引脚（sys_config.fex）

| 信号 | GPIO |
|---|---|
| CLK | PC02 |
| CS | PC03 |
| MOSI | PC04 |
| MISO | PC05 |
| WP | PC06 |
| HOLD | PC07 |

### 分区表（sys_partition_nor.fex）

| 分区 | 扇区数 | 约大小 | 作用 |
|---|---|---|---|
| boot-resource | 256 | 128 KB | 开机 logo / 资源 |
| env | 512 | 256 KB | U-Boot 环境变量 |
| boot | 8704 | 4.25 MB | 内核 + DTB（`boot.fex`） |
| rootfs | 20480 | **10 MB** | squashfs 只读系统 |
| rootfs_data | 768 | 384 KB | 数据区 |
| UDISK | 剩余 | ~17 MB | OpenWrt overlay 可写层 |

换算：`1 扇区 = 512 字节`。32MB Flash 总扇区 65536，固定分区之外自动分给 UDISK。

启动参数（env.cfg）：

- NOR 根文件系统：`nor_root=/dev/mtdblock4`
- 文件系统类型：`rootfstype=squashfs`
- 控制台：`console=ttyS0,115200`

> 若后续扩大 rootfs（如文档中 23MB 方案），需同步改 `sys_partition_nor.fex` 并重新打包，详见 Ubuntu `H133-rootfs空间与应用占用说明.md`。

## 显示子系统

### 双屏输出（board.dts disp 节点）

| 屏幕 | output_type | 模式 | 用途 |
|---|---|---|---|
| screen0 | 1（LCD） | mode 4 | **MIPI 主屏** |
| screen1 | 3（HDMI） | mode 10（1080p60） | **HDMI 外接** |

### Framebuffer

| 项 | 值 |
|---|---|
| fb0 分辨率 | **800 × 1280**（竖屏 UI） |
| fb0 格式 | RGB（format=0） |
| HDMI 模式检测 | `hdmi_mode_check = 1` |

### MIPI 面板（lcd0，当前启用）

| 项 | 值 |
|---|---|
| 接口 | MIPI DSI（`lcd_if = 4`） |
| 分辨率 | **800 × 1280** |
| 物理尺寸 | 135 × 216 mm（约 10.1 寸竖屏） |
| DSI 模式 | Video mode（`lcd_dsi_if = 0`） |

工程内另有 LVDS 1280×800 面板配置段（`lcd_if = 3`），本板 p1_nor 以 **MIPI 800×1280** 为准。

### HDMI

- 输出模式：1080p60（screen1_output_mode = 10）
- 色彩：BT709，HDMI mode（非 DVI）
- 支持 CEC / ARC（应用层 `ENABLE_CEC`、`ENABLE_ARC`）

## 输入与人机

### 电容触摸（当前板）

| 项 | 值 |
|---|---|
| 驱动 | **gslX680**（`compatible = "allwinner,gslX680"`） |
| 总线 | TWI1，地址 0x40 |
| 触摸 RAW | 1280 × 800（横） |
| LVGL 坐标 | 800 × 1280（竖）——在 **evdev 层 swap**，非驱动内改 |

> 其他硬件版本可能用 GT911，见 `H133开发文档-wjz/驱动/TP驱动&鼠标&触摸/`；**本工程 p1_nor 为 gslX680**。

### 其他输入

- USB HID 鼠标（与触摸并存）
- 按键 / IR（见 board.dts gpio-keys）

## 有线接口

| 接口 | 配置要点 |
|---|---|
| UART0 调试 | PB08(TX) / PB09(RX)，115200 |
| UART3 扩展 | PG0/PG1（P15 排针，与 sdc1 互斥） |
| SD 卡 card0 | PF0–PF5 四线，CD=PF6 |
| USB | OTG + Host（adb / WiFi  dongle） |
| 音频 | SPDIF + 功放 UART 控制 |

SD 卡挂载点 `/mnt/SDCARD`，由应用 DiskManager 管理，**不由 fstab 自动挂载**。

## 无线

board.dts 含 Allwinner rfkill 节点：

- `sunxi-wlan` — WiFi
- `sunxi-bt` — 蓝牙
- `sunxi-btlpm` — BT 低功耗

具体模组型号依 PCB 版本，见 WiFi 驱动文档。

## 板级配置文件

```
device/config/chips/h133/configs/p1_nor/
├── sys_config.fex        # GPIO、DRAM、UART、SPI NOR、SD
├── board.dts             # Linux：显示、触摸、WiFi、音频
├── uboot-board.dts       # U-Boot 阶段显示
├── sys_partition_nor.fex # Flash 分区
└── env.cfg               # 内核 cmdline 模板
```
