# H133 项目基本信息

## 项目定位

| 项 | 说明 |
|---|---|
| 产品形态 | 便携式投影仪 / KTV 点播一体机 |
| 目标硬件 | 拓步 H133-10.1 寸方案（MIPI 竖屏 + HDMI 外接） |
| 软件定位 | Tina Linux 嵌入式 Linux，非 RTOS |
| 主应用 | `lv_projector`——基于 LVGL 8.3.2 的全功能 UI 与业务层 |
| 维护状态 | 活跃开发（Ubuntu 工程最后编译约 2026-06） |

## 核心标识

| 项 | 值 | 来源 |
|---|---|---|
| SoC | Allwinner **H133** | board.dts `compatible = "allwinner,h133"` |
| 平台代号 | sun8iw20p1 | 内核 / U-Boot |
| 板型目录 | **p1_nor** | `device/config/chips/h133/configs/p1_nor/` |
| machine 名 | **p1** | sys_config.fex `[product] machine = "p1"` |

> 命名说明：应用里版本前缀写 `H135-P1-`，芯片仍为 H133，属产品命名习惯，以 `projector_config.h` 为准。

## 板型对照

工程在 `device/config/chips/h133/configs/` 提供多套板级配置：

| 目录 | 启动方式 | 使用场景 |
|---|---|---|
| **p1_nor** | SPI NOR | **当前主用**，拓步量产板 |
| p1 | eMMC/SD 变体 | 同硬件不同 Flash 方案 |
| p2 / p2nor | NOR/eMMC | 另一 PCB 版本，路径中替换 `p1_nor` 即可 |
| ft | — | 工厂测试 |
| default | — | SDK 模板 |

切换板型 = 换 `configs/<板型>/` + `openwrt/target/h133/h133-<板型>/` + 重新 `m && p`。

## 功能模块一览

### 显示与 UI

- **screen0**：MIPI LCD，fb0 = **800×1280**（竖屏 10.1 寸）
- **screen1**：HDMI，默认 **1080p60**
- G2D 合成 / sunxifb 送显；支持软旋转、双显异显（见驱动文档）

### 多媒体与业务

- 本地 / 网络媒体（CedarX / TPlayer）
- KTV 点播、MQTT、豆包 AI 语音
- 页面导航栈 + GUI Guider 生成 UI

### 投屏与连接

| 功能 | 宏开关（projector_config.h） |
|---|---|
| Miracast | `ENABLE_MIRACAST = 1` |
| DLNA | `ENABLE_DLNA = 1` |
| AirPlay | `ENABLE_AIRPLAY = 1` |
| WiFi | `ENABLE_WIFI = 1`（依赖 `LV_USE_LINUX_WIFI2_MODE`） |
| 蓝牙 | `ENABLE_BT = 1`（依赖 `LV_USE_LINUX_BT4_MODE`） |

### 外设与存储

- 触摸：**gslX680**（I2C TWI1，非 GT911——本板 p1_nor 实测）
- USB 鼠标可并存（evdev 层坐标映射）
- SD 卡 → `/mnt/SDCARD`（**DiskManager** 独占管理，`ENABLE_DISK_MANAGER = 1`）
- 功放板 UART + SPDIF（拓步方案）

### 调试与交付

- UART0 115200（PB8/PB9）
- OpenSSH/SFTP（root/tina:22）
- adb push 快速刷 `lv_projector`
- 整包烧录 `out/h133_linux_p1_nor_uart0_nor.img`
