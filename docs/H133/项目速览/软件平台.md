# H133 软件平台

> 板型：**p1_nor** · 应用：**lv_projector 8.3.2**

## 软件栈总览

```
┌──────────────────────────────────────────────────────┐
│  lv_projector 8.3.2                                   │
│  KTV · 豆包 · MQTT · 投屏 · 设置 · 媒体 · 工厂测试     │
├──────────────────────────────────────────────────────┤
│  LVGL 8.3 + sunxifb + evdev + GUI Guider 生成 UI      │
├──────────────────────────────────────────────────────┤
│  中间件：CedarX/TPlayer · DiskManager · AWCast       │
├──────────────────────────────────────────────────────┤
│  Tina rootfs（BusyBox init，无 procd）                 │

├──────────────────────────────────────────────────────┤
│  Linux 5.4 + 全志 disp/g2d/mmc/wifi 驱动              │
├──────────────────────────────────────────────────────┤
│  U-Boot（brandy-2.0）                                 │
└──────────────────────────────────────────────────────┘
```

## 版本与构建

| 组件 | 版本 / 说明 |
|---|---|
| 内核 | Linux **5.4**（`kernel/linux-5.4/`） |
| U-Boot | brandy-2.0 |
| LVGL / lv_projector | **8.3.2**（OpenWrt PKG_VERSION） |
| 固件版本前缀 | `H135-P1-` |
| rootfs 格式 | **squashfs** 只读（`/dev/mtdblock4`） |
| 可写层 | UDISK overlay（~17MB） |
| 宿主机构建 | Ubuntu 18.04 x86_64，gcc 7.5.0 |

## 启动链

```
SPI NOR boot
  → U-Boot（读 env 分区）
  → setargs_nor：root=/dev/mtdblock4 rootfstype=squashfs
  → Linux 5.4（console=ttyS0,115200）
  → /init（BusyBox）
  → rc.final → sshd_start.sh
  → /usr/bin/lv_projector
```

关键 env.cfg 变量：

| 变量 | 值 |
|---|---|
| console | ttyS0,115200 |
| nor_root | /dev/mtdblock4 |
| rootfstype | squashfs |
| bootcmd | run setargs_nor boot_normal |
| cma | 8M |

## 显示与输入软件链

```
lv_projector UI (800×1280)
  → sunxifb (fb0)
  → G2D blit / blend
  → DE → MIPI LCD (screen0) + HDMI (screen1)

触摸 gslX680 → evdev (坐标 swap) ─┐
USB 鼠标 → evdev ─────────────────┼→ LVGL indev
```

G2D 用户态头文件必须与内核 `g2d_driver.h` **ABI 一致**，且须在工程顶层编译，见 [[../踩坑记录/index]]。

## rootfs 与空间（参考）

| 指标 | 数值 | 说明 |
|---|---|---|
| rootfs 分区上限 | 10 MB（当前 fex） | 可随分区方案扩大 |
| squashfs 镜像 | ~10–21 MB | 随安装包变化 |
| 整包 .img | ~17 MB（2026-06 构建） | 含 boot + rootfs |
| 应用二进制 | `/usr/bin/lv_projector` | adb/scp 快速替换 |

详细占用分析：Ubuntu `H133开发文档-wjz/基础资料/H133-rootfs空间与应用占用说明.md`

## OpenWrt 板级包配置

```
openwrt/target/h133/h133-p1_nor/
├── defconfig                    # 包选型
├── busybox-init-base-files/
│   ├── etc/init.d/rc.final
│   └── usr/bin/sshd_start.sh
└── base-files/
```

主应用 OpenWrt 包：

```
openwrt/package/thirdparty/gui/lvgl-8/lv_projector/Makefile
  PKG_NAME:=lv_projector
  PKG_VERSION:=8.3.2
```
