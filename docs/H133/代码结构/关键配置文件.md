# H133 关键配置文件

## 板级硬件配置

| 文件 | 路径 | 作用 |
|---|---|---|
| sys_config.fex | `device/config/chips/h133/configs/p1_nor/sys_config.fex` | GPIO、UART、SPI NOR、SD、DRAM |
| board.dts | 同上 `board.dts` | Linux 设备树（外设节点） |
| uboot-board.dts | 同上 | U-Boot 设备树 |
| sys_partition_nor.fex | 同上 | NOR 分区表 |
| env.cfg | 同上 | U-Boot 环境变量 |

> 修改以上文件后必须：`./build.sh kernel bootloader && m && p`

## OpenWrt / rootfs

| 文件 | 路径 | 作用 |
|---|---|---|
| defconfig | `openwrt/target/h133/h133-p1_nor/defconfig` | 包选型（openssh 等） |
| rc.final | `.../busybox-init-base-files/etc/init.d/rc.final` | 开机脚本 |
| sshd_start.sh | `.../busybox-init-base-files/usr/bin/sshd_start.sh` | SSH 启动 |

## 应用配置

| 文件 | 路径 | 作用 |
|---|---|---|
| projector_config.h | `lv_projector/src/include/projector_config.h` | 功能开关宏 |
| lv_conf.h | `lv_projector/src/lv_conf.h` | LVGL 配置 |
| lv_drv_conf.h | `lv_projector/src/lv_drv_conf.h` | 显示/输入驱动配置 |
| sys_param | `lv_projector/src/sys_param.*` | 运行时参数持久化 |

## G2D ABI 相关（踩坑高频）

| 文件 | 说明 |
|---|---|
| `kernel/linux-5.4/include/linux/g2d_driver.h` | **内核侧** G2D 结构体（权威） |
| `lv_projector/.../g2d_driver_enh.h` | 用户态头文件，**不可多字段** |
| build_dir 内旧头文件 | OpenWrt 可能缓存旧版，需 `make clean` 或删 build_dir |

## 编译环境变量

通过 `source build/envsetup.sh` 注入，典型：

- `LICHEE_BOARD` = p1_nor
- `LICHEE_CHIP` = h133
- `LICHEE_BR_OUT` = out/h133/p1_nor/openwrt/...

**禁止**在 `openwrt/openwrt/` 子目录单独 make，会导致上述变量缺失。
