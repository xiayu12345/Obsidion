# H133 应用层结构

## 架构概览

`lv_projector` 在 LVGL 8 之上按**功能域**划分模块，GUI Guider 生成 UI 骨架，业务逻辑在各子目录实现。

```
┌──────────────────────────────────────────┐
│  UI 层：generated/ + gui_guider + 各 Activity │
├──────────────────────────────────────────┤
│  业务层：Layer/ KTV / 豆包 / MQTT / 导航栈   │
├──────────────────────────────────────────┤
│  服务层：Media / Network / System / Common  │
├──────────────────────────────────────────┤
│  平台层：LVGL + sunxifb + evdev + CedarX    │
└──────────────────────────────────────────┘
```

## 模块说明

| 目录 | 职责 |
|---|---|
| `lv_pro_launcher.c` | 主桌面、应用入口 |
| `Layer/` | KTV 点播、豆包 AI、MQTT、页面导航栈 |
| `Media/` | 本地/网络媒体播放，middle_ware 解码封装 |
| `Setting/` | 系统/图像/声音/ Keystone 设置 |
| `Network/` | WiFi、网络状态 |
| `Bluetooth/` | 蓝牙连接与资源 |
| `AWCast/` | DLNA、Miracast（libwfd2、wfd_player） |
| `USBCast/` | USB 投屏 |
| `WiredSP/` `WirelessSP/` | 有线/无线同屏 |
| `Common/disk_manager/` | SD 卡挂载与热插拔 |
| `Common/setting/` | 设置公共逻辑 |
| `System/` | system_api、底层系统调用封装 |
| `Factory_test/` | 产测（HDMI/WiFi/BT/传感器） |
| `generated/` | GUI Guider 自动生成（勿手改） |
| `widget/` | 自定义 LVGL 控件 |

## 页面导航

- `page.h` / `page_ui.h` — 页面 ID 定义
- `lvgl_page_navigation.c` — 页面切换
- `page_nav_stack.c` — 导航栈（返回逻辑）

## 多媒体链路

```
媒体文件 / 网络流
  → lv_pro_res_media_player_int（middle_ware）
  → CedarX / TPlayer
  → DE / G2D → HDMI / MIPI
```

## 扩展新功能建议

1. 在 `include/` 增加头文件与 page ID
2. 业务代码放 `Layer/` 或独立子目录
3. UI 优先 GUI Guider 生成到 `generated/`
4. 在 `lv_pro_launcher` 或导航栈注册入口
