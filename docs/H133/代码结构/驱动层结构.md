# H133 驱动层结构

## 分层

```
用户态应用 (lv_projector)
    ↓ ioctl / fb / evdev / ALSA
Linux 5.4 内核驱动
    ↓
设备树 board.dts + sys_config.fex
    ↓
H133 硬件外设
```

## 内核驱动（常见）

| 子系统 | 内核路径 / 节点 | 说明 |
|---|---|---|
| 显示 DE | `drivers/video/sunxi/disp2/` | 显示引擎，fb0 |
| G2D | `drivers/char/sunxi_g2d/` | 2D 合成/旋转，与 sunxifb 配合 |
| MMC/SD | `drivers/mmc/` | mmcblk0 → /mnt/SDCARD |
| HDMI | DRM/HDMI 相关 | 详见 HDMI 驱动文档 |
| MIPI DSI | LCD 面板驱动 | 10.1 寸屏参数在 DTS |
| I2C / GT911 | `drivers/input/touchscreen/` | 触摸 |
| USB | DWC2 / HID | 鼠标、U 盘 |
| WiFi | 模组相关驱动 | USB 或 SDIO |
| UART | `drivers/tty/serial/` | 调试 serial0 = PB8/PB9 |
| 音频 | ALSA / SPDIF | 功放板 UART 控制 |

## 用户态驱动封装

| 组件 | 路径 | 作用 |
|---|---|---|
| sunxifb | `lv_drivers/display/sunxifb.c` | LVGL 帧缓冲 + G2D blit |
| evdev | `lv_drivers/indev/evdev.c` | 触摸/鼠标输入 |
| g2d_driver_enh.h | lv_projector 内 | G2D ioctl 封装（**须与内核 ABI 一致**） |
| DiskManager | `Common/disk_manager/` | SD netlink 用户态管理 |

## 设备树关键节点

```
board.dts (p1_nor)
├── &uart0          # 调试串口 PB8/PB9
├── &uart3          # P15 排针 PG0/PG1
├── &mmc0 / card0    # SD 卡 PF0-PF5
├── &spi0           # SPI NOR PC2-PC7
├── &lcd / &hdmi    # 显示输出
└── &twi0           # I2C（触摸 GT911）
```

## 调试入口

```bash
# 板端
cat /proc/mounts
ls -l /dev/mmcblk*
cat /sys/class/disp/disp/attr/sys
dmesg | grep -E 'G2D|DISP|mmc|hdmi'
```
