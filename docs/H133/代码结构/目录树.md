# H133 工程目录树

> 路径基准：`~/Workspace/HDMI_new/HDMI`

```
HDMI/
├── build/                    # Tina 构建入口
│   ├── envsetup.sh           # source 加载环境
│   ├── mkcommon.sh / quick.sh
│   └── pack/                 # 打包工具
├── brandy/brandy-2.0/        # U-Boot 源码
├── kernel/linux-5.4/         # Linux 5.4 内核
│   ├── arch/arm/boot/dts/    # 设备树（板级链接自 device/）
│   └── drivers/              # 厂商驱动（display/g2d/mmc 等）
├── device/config/chips/h133/configs/
│   └── p1_nor/               # ★ 当前板级配置
│       ├── board.dts
│       ├── sys_config.fex
│       └── sys_partition_nor.fex
├── openwrt/
│   ├── openwrt/              # OpenWrt 核心
│   ├── package/              # 软件包（含 lv_projector）
│   └── target/h133/h133-p1_nor/  # ★ 板级 OpenWrt 配置
├── platform/
│   ├── allwinner/            # 厂商中间件
│   └── thirdparty/gui/lvgl-8/
│       ├── lvgl/             # LVGL 库
│       └── lv_projector/     # ★ 主应用
│           └── src/          # 业务源码
├── prebuilt/                 # 预编译 host/target 工具
├── tools/                    # 烧录/签名工具
├── docs/                     # 工程 Markdown 笔记
├── H133开发文档-wjz/         # 完整文档归档
└── out/
    ├── toolchain/            # 交叉编译工具链
    └── h133/p1_nor/          # ★ 编译产物
        └── openwrt/build_dir/target/lv_projector/
```
