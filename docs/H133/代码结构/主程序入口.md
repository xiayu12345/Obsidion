# H133 主程序入口

## 入口文件

```
platform/thirdparty/gui/lvgl-8/lv_projector/src/main.c
```

## 启动流程（概要）

```
main()
  ├── LVGL / sunxifb 显示初始化
  ├── evdev 输入初始化（触摸/鼠标）
  ├── system_init_early()          # ★ 含 DiskManager_Init（SD 卡）
  ├── wait_sd_mount_ready(2)       # 开机 SD 探测（最多 2 秒）
  ├── sys_param / projector_config 加载
  ├── lv_pro_launcher 启动桌面
  ├── 各业务模块初始化（KTV/MQTT/豆包/投屏…）
  └── lv_task_handler 主循环
```

## 关键初始化

| 函数 / 模块 | 作用 |
|---|---|
| `system_init_early()` | 早期系统 init，**必须调用**否则 SD 不挂载 |
| `wait_sd_mount_ready()` | 等待 `/mnt/SDCARD` 挂载完成 |
| `DiskManager_Init()` | netlink uevent 监听 SD 热插拔 |
| `lv_pro_launcher` | 主桌面 UI |
| `gui_guider` / `events_init` | GUI Guider 生成界面 |

## SD 卡相关（重要）

工程策略：**SD 只由 DiskManager 管理**，`hotplug.sh` / `mdev` / `fstab` 不自动挂 SD。

若 `/dev/mmcblk0*` 存在但 `/mnt/SDCARD` 为空，检查 `main.c` 是否注释了 `system_init_early()`。
